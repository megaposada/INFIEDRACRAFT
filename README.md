# INFORMACION

- Infiedracraft fue una serie hardcore **creada por megaposada** que inicio el **1 de agosto de 2022** y finalizo el **27 de agosto de 2022**, despues de su salida se realizo este addon que mejora en varios aspectos la jugabilidad y la experiencia al jugar el addon
-  Cualquier intento de hacer pasar el addon como tuyo sera castigado, tienes permiso de tomar prestado parte de su codigo siempre y cuando pidas permiso y des creditos

# DESCARGAS

- El addon aun no esta disponible

# TUTORIAL DEL ADDON

- Para revivir a un jugador deberas poner **/execute as "nombredeljugador" run function revivir**
- El addon ya viene con kits para eventos incluidos, para darselo a todos los jugadores debes colocar: **/function kits, /function kits2**

(El kits es de mob arena/dragon y el kits2 es de armor farming)
